# TypeScript Student Interface Project

This project demonstrates TypeScript interface creation and DOM manipulation.

## Project Structure

- `task_0/js/main.ts` - Main TypeScript file containing Student interface and table creation
- `task_0/package.json` - Project dependencies and scripts
- `task_0/tsconfig.json` - TypeScript configuration
- `task_0/webpack.config.js` - Webpack bundling configuration
- `task_0/.eslintrc.js` - ESLint configuration

## Features

- Student interface with firstName, lastName, age, and location properties
- Two student instances created
- Vanilla JavaScript table generation
- TypeScript type checking
- Webpack bundling

## Usage

```bash
cd task_0
npm install
npm run build
```

The build will create a `dist/` directory with the bundled JavaScript and HTML files.
